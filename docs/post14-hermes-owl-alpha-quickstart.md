## Setup SOP

### Phase 1 — Set Up OpenRouter (10 min)
1. Go to openrouter.ai
2. Sign up (Google or email)
3. Profile → API Keys → Create Key
4. Name it "Hermes Agent Key" and copy it
5. Search "Owl Alpha" — confirm $0/M tokens
6. Note the model string: `openrouter/owl-alpha`

### Phase 2 — Install Hermes Agent (15 min)
1. Download from Nous Research
2. Install for your OS (Mac/Windows/Linux)
3. Settings → Model Provider → OpenRouter
4. Paste API key
5. Model: `openrouter/owl-alpha`
6. Test connection

### Phase 3 — Run First Task (10 min)
1. Pick one weekly task (30+ min)
2. Type it in plain English
3. Watch the agent work
4. Review and refine

### Phase 4 — Build First Workflow (30 min)
1. Pick a multi-step process
2. Break into individual steps
3. Create prompts for each step
4. Chain outputs together
5. Save as workflow
6. Schedule automation

### Phase 5 — Scale (Ongoing)
1. Add one new workflow per week
2. Document what works
3. Stack workflows so agents feed each other
4. Audit every 30 days
