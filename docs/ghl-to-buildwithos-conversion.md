## Overview

BuildwithOS is the most AI-forward Lead Connector derivative. It has a fundamentally different approach: instead of traditional funnels, it uses **pre-built business kits** and **AI agents** for lead capture and conversion.

**Key Difference:** BuildwithOS doesn't have a direct "funnel builder" like GoHighLevel. Instead, it provides:
- Pre-built high-converting funnel templates (installed instantly)
- AI voice and chat agents for lead qualification
- AI phone calls for outbound/inbound (Platinum plan)
- Project-based ontology for managing work

## Template Mapping

### 1. Book a Call Funnel
| | GoHighLevel | BuildwithOS |
|---|---|---|
| **Purpose** | Book appointments | Book appointments via AI |
| **Pages** | Opt-in → Booking → Confirmation | AI Agent qualifies → Calendar books → Confirmation |
| **How** | "Add funnel" → "Book a Call" | AI voice/chat agent + Google Calendar sync |
| **Key Difference** | Manual form | AI agent qualifies leads first |

**BuildwithOS Equivalent Setup:**
1. Set up Google Calendar integration (Profile → Integrations)
2. Configure AI agent for lead qualification
3. Set up AI phone call or chat agent to book calls
4. Connect calendar for automatic booking
5. Set up confirmation email/SMS

### 2. ChatGPT Opt-In Funnel
| | GoHighLevel | BuildwithOS |
|---|---|---|
| **Purpose** | Capture leads with AI resource | Capture leads with AI agent |
| **Pages** | Opt-in → Delivery → Thank You | AI Chat Agent → Delivery → Thank You |
| **How** | "Add funnel" → "ChatGPT Opt In" | AI chat widget on landing page |
| **Delivery** | Email with prompts | AI agent delivers + email follow-up |

**BuildwithOS Equivalent Setup:**
1. Create landing page with AI chat widget
2. Configure AI chat agent to qualify and capture
3. Set up automation: lead captured → deliver resource via AI agent
4. Connect email for follow-up sequence
