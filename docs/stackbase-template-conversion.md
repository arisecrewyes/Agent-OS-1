## Critical Context: GoStackBase Template Sharing

**Unlike GoHighLevel, GoStackBase does NOT have a public funnel sharing system.** GoHighLevel uses `funnel_share` links that let you import any template with one click. GoStackBase templates ("15-Minute Setup Kits") are only accessible **inside the platform dashboard** after you log in.

This means:
- ❌ There are NO public GoStackBase funnel share links equivalent to GHL's `affiliates.gohighlevel.com/?funnel_share=...` system
- ✅ GoStackBase provides equivalent functionality through its built-in template library
- ✅ You can recreate any GHL funnel in GoStackBase using the steps below
- ✅ GoStackBase's "15-Minute Setup Kits" are the closest equivalent to GHL's pre-built funnels

## The 5 GHL Funnel Templates → GoStackBase Equivalents

### 1. Book a Call Funnel

| | GoHighLevel | GoStackBase |
|---|---|---|
| **GHL Link** | [Book a Call Funnel](https://affiliates.gohighlevel.com/?fp_ref=goldie-agency54&funnel_share=65914bbf139166048fcfe1ba) | Use 15-Minute Setup Kit inside dashboard |
| **Purpose** | Book appointments/calls | Book appointments/calls |
| **Pages** | Opt-in → Booking → Confirmation | Smart Form → Calendar → Thank You |
| **GHL How** | "Add funnel" → "Book a Call" | Landing Page Builder + Smart Calendar |
| **Equivalent Setup** | — | Smart Form → Calendar → Thank You page |

**GoStackBase Steps:**
1. Log in → https://gostackbase.com/login
2. Go to "Landing Pages" → Create New
3. Add Smart Form (name, email, phone)
4. Connect Smart Calendar for booking
5. Set up confirmation/thank-you page
6. Configure automation: form submit → book appointment → send confirmation email
