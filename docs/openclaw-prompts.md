## Content Creation

### SEO Blog Post
```
Create an SEO-optimized blog post about [topic],
include keyword [keyword] 5-7 times naturally,
add internal links to [your site],
and publish to WordPress
```

### YouTube Thumbnail
```
Create a YouTube thumbnail about [topic]
```

### AI Avatar Video
```
Create an AI avatar video about [topic] using HeyGen
```

### Voice Note
```
Create a voice note in my voice about [topic]
```

### Landing Page
```
Create a landing page for [business],
make it conversion-optimized,
deploy to Netlify at [domain]
```

### Notion Page
```
Create a Notion page about [topic]
with step-by-step instructions
```

## Research & Monitoring

### YouTube Competitor Monitoring
```
Monitor these competitor YouTube channels: [URLs].
Alert me when they publish a video that performs 2x better
than their average. Check every 60 minutes.
```

### Trending Topic Research
```
Search Reddit and Twitter for the latest trending topics
about [niche]. Give me the top 5 topics with a summary
of each and why they're trending.
```

### Web Search
```
Using the Chrome extension, go to Google and search for [query].
Give me the top 3 results with summaries.
```

## Workflows

### Complete SEO Workflow
```
Research keyword → Generate outline → Write 1500-word blog post →
Optimize for SEO → Add internal links → Publish to WordPress →
Create social media posts → Share on Twitter/X
```

### Content Pipeline
```
Watch folder: /content/drop
When new files appear (PDFs, notes, transcripts),
scan them and group by topic.
Each morning, send me the top 3 content ideas
with titles and hooks.
```

### Lead Management
```
Scan my inbox every 30 minutes for new leads.
Extract: name, company, need, timeline, budget.
Add to Google Sheet.
Draft a personalized response.
Send me Telegram message with lead details.
I'll approve the response, it sends.
```

## Tips for Better Prompts

1. **Be specific** — Include the exact topic, keywords, and desired outcome
2. **Include context** — Tell OpenClaw about your brand, audience, and goals
3. **Chain tasks** — Combine multiple steps into one prompt for complex workflows
4. **Iterate** — If the first result isn't perfect, refine the prompt with feedback
5. **Save winners** — When you find a prompt that works well, save it as a template
