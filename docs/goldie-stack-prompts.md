1. Copy any prompt below into Hermes Agent (or any connected AI tool)

3. The more context in your memory file, the better the results
4. Give feedback after every task — this is how the system improves

## 🔁 RECAP: THE 3-STEP AUTOMATION BLUEPRINT

Every automation follows this same process:

**Step 1** — Where do you spend your time? Identify the repetitive task.

**Step 2** — What is the one thing you can automate this week? Pick one specific task.

**Step 3** — Ask Hermes:
```
Based on what you know about me, what could you help me automate day-to-day specifically for me?
```

Then: Define the trigger → Give it the data sources (APIs, credentials) → Test it once → Schedule it.

---

## KEY PRINCIPLES

1. **The more you use it, the better it gets.** This is a compounding effect.
2. **Always give feedback on every task.** This is how it improves.
3. **String multiple skills together.** Skill 1 (research with Firecrawl) + Skill 2 (post to X) + Skill 3 (generate image with Nana Banana) = a full automated content pipeline.
4. **Create your own skills** rather than only relying on community ones. Custom skills are tailored exactly to your workflow.
5. **Back up your skill.md files weekly.**
6. **Be prepared for the first 7 days.** It will not be perfect but it gets dramatically better by day 7.
7. **If posting to social media, always ask for a draft first and approve before posting.** Do this at least 10 times before letting it post autonomously.
