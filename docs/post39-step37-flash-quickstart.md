## Step-by-Step Setup

### Step 1: Install Hermes Agent
```bash
# Clone and install
git clone https://github.com/crawlab-team/hermes
cd hermes
# Follow README install instructions
```

### Step 2: Get OpenRouter API Key
1. Go to [openrouter.ai](https://openrouter.ai)
2. Sign up (free)
3. Navigate to **Keys** → Create API key
4. Copy the key

### Step 3: Connect Step 3.7 Flash
1. Open Hermes Agent
2. Go to **Settings** → **Model**
3. Select **OpenRouter** → **Step 3.7 Flash**
4. Paste your OpenRouter API key
5. Save

### Step 4: Connect Messaging (Optional)
- Telegram: Link your Telegram account
- Discord: Link your Discord
- WhatsApp: Link your WhatsApp

### Step 5: Test It
```
Hello! Can you search the web for the latest AI news and give me 3 bullet points?
```
