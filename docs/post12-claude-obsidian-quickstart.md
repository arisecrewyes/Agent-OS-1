## Problem & Solution

**Problem:** Every AI agent starts from zero context each chat. Generic answers that apply to anyone.

**Solution:** Build a second brain that every agent reads from and writes to. Compounds daily.

## The Three Free Tools

| Tool | Role | URL |
|------|------|-----|
| **OMI** | Capture layer — background app, auto-captures notes on conversations/decisions/tasks | https://www.omi.me |
| **Obsidian** | Storage layer — markdown file vault with knowledge graph. Universal memory every AI can read. | https://obsidian.md |
| **Claude** | Reader + organizer — reads vault before answering, writes useful notes back | https://claude.ai |

## The Two-Way Loop

```
OMI captures your day → Obsidian stores it
         ↓                               ↓
Claude reads vault → Contextualized answers
         ↓
Claude writes notes back → Vault grows smarter
         ↓
Repeat daily — compounds over months
```

## PARA Folder Structure

| Bucket | Contents |
|--------|----------|
| **P** — Projects | Active work, current quarter goals, live campaigns |
| **A** — Areas | Ongoing responsibilities: marketing, clients, team |
| **R** — Resources | Reference: books, frameworks, articles, tools, prompts |
| **A** — Archive | Inactive items (still searchable, out of the way) |

## Key Prompts (Copy-Ready)

### Getting Started
```
I just created an Obsidian vault. Help me create a PARA folder structure for a solopreneur who runs an online business. Give me the exact folders to create and what goes in each one.
```

### Daily Workflow
```
Here are today's raw OMI memory notes [paste them]. Clean these up into a structured daily log for my Obsidian vault. Highlight any decisions made, tasks to follow up on, and tools I used.
```

```
Good morning. Based on my Obsidian vault context [paste relevant notes], give me a morning briefing. Include: my top priority today, open loops from yesterday, any decisions I need to make, and the most important thing I should focus on in the next 90 minutes.
```

```
Here are my daily notes from this week [paste them]. Run a weekly review. Give me: top 3 wins, top 3 problems, biggest lessons, what to carry forward next week, and what to archive.
```

### Building Systems
```
Based on my business context [paste your About Me document], create 20 high-value prompts I should save in my Obsidian vault under a Prompt Library folder. These should be prompts I will use repeatedly for content creation, business decisions, client work, and automation ideas.
```

```
Create a constraint tracking template for my Obsidian vault. It should track: the constraint identified, what I tried to fix it, what worked, what did not work, and the date resolved.
```

```
Create a wins and testimonials tracker for my Obsidian vault. Every win should be logged with: date, what happened, what led to it, and how replicable it is. Give me the markdown template.
```

### Context Wiring (paste at start of every AI chat)
```
Before answering anything, here is my full context: [paste About Me + current project notes]. Use this as your operating context for this entire conversation.
```

### Vault Audit
```
Here is a summary of my Obsidian vault structure [describe it or paste your folder names and note titles]. Audit this second brain. What is missing? What connections am I not making? What should I add in the next two weeks to make this more powerful?
```

### Agent OS Design
```
Based on my Obsidian vault [paste context], design an agent operating system for my business. Which tasks should be handled by Claude? Which by a research agent? Which by a content agent? Map out the workflow and which vault notes each agent should read before starting.
```

## Core Belief Shifts

| Old Belief | New Belief |
|------------|------------|
| AI gives generic answers, not that useful | AI quality = context quality. Give it a brain, changes the category of tool |
| Too complicated to set up | Just 3 free tools passing notes. Simpler than email. |
| Not enough notes to start | 10 well-written notes give more context than 100 blank conversations. Start small. |
| Platforms will add memory eventually | Even if they do, it is siloed to one platform. Obsidian belongs to you, works with every agent. |
| No time to maintain it | OMI writes notes. Claude organizes. You work. 10 min/week maintenance. |
| Not technical enough | If you can organize a desktop folder, you can build this. No code required. |
| Won't change AI performance | Categorical difference, not incremental. Smart stranger becomes smart colleague. |
| Will lose everything if app shuts down | Obsidian = plain text files on your machine. Survives every platform. |

---

## Why This Matters for All AI Builders

Any AI agent (GoStackBase, Claude, Gemini, etc.) benefits from persistent context. An Obsidian vault serves as the **universal memory layer** — the vault is on your machine, readable by every tool, owned by you.

**Maintenance:** 10 minutes/week (Sunday). Export OMI → Claude processes weekly review → archive outdated notes. The rest is automated.
