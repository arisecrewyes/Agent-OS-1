## How to Use These SOPs

1. **Open each Google Doc** — Request access if needed (they may be public or require sign-in)
2. **Copy the templates** — Each doc has copy-paste ready templates
3. **Customize for your niche** — Swap out the examples with your own offers
4. **Automate** — Use n8n, OpenClaw, or scheduling tools to automate the posting workflows
