2. **No video generation** — images only
3. **No voice/audio** — visual only
4. **No batch processing** — one image at a time
5. **No direct social media integration** — no auto-post
6. **No WordPress integration** — no auto-publish
7. **No webhook/callback** — no automation triggers
8. **No collaboration** — single user session
9. **No custom model training** — only pre-built models
10. **No desktop app** — browser only
