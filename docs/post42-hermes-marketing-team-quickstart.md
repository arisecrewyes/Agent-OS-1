## The Goldie Omnipresence Stack — 6 Layers

| Layer | Name | What It Does |
|-------|------|-------------|
| 1 | **SEO Content Stack** | 1 keyword → 5 articles → published across 5 sites |
| 2 | **Visual Authority Stack** | Auto-generates custom images via Grok 4.3 for every article |
| 3 | **Hyperframes Video Stack** | Writes script → voiceover → animated slides → finished video |

---

## Layer Details

### Layer 1: SEO Content Stack
- 1 keyword → 5 optimized articles
- Published across 5 different websites automatically
- **Secret weapon:** Real case studies make content unique (not generic AI fluff)

### Layer 2: Visual Authority Stack
- Connects to **Grok 4.3** for custom images
- Every article gets a custom image
- Custom social media images
- Ranks on Google Image Search

### Layer 3: Hyperframes Video Stack
- Free, open-source skill inside Hermes Agent OS
- Agent writes script → generates voiceover → builds animated slides → renders video
- Rank with video AND article for the same keyword simultaneously

### Layer 4: Anti-Gravity Website Builder Stack
- **Anti-Gravity 2.0** — free AI agent toolkit
- Builds CSS, HTML, content structure
- Delivers ready-to-deploy in minutes
- Niche websites, landing pages, full blogs

### Layer 5: Agent Task System (Kanban Board)
- Assign one big project
- System breaks into tasks → assigns to different agents
- All agents work simultaneously
- Entire blog/strategy/social media plan done while you're away

### Layer 6: Infinite Context Engine
- Agents take notes on you all day
- Learns brand, voice, results, case studies
- Knowledge graph inside Obsidian
- 1,553+ individual memories loaded
- Gets smarter every day

## Key Prompts

### Case Study Injection
```
Here is a case study from my business: [paste case study].
Use this to write 5 unique articles. Each must include specific data points from the case study.
```

### Multi-Site Publishing
```
Take this article and publish it to all 5 sites:
- site1.com/blog/[slug]
- site2.com/blog/[slug]
- site3.com/blog/[slug]
- site4.com/blog/[slug]
- site5.com/blog/[slug]
```

### Video + Article Combo
```
For the keyword [keyword]:
1. Write a full SEO article
2. Create a video script from the article
3. Generate a voiceover
4. Build animated slides
5. Render the final video
```
