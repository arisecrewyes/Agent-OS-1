## Overview

HighLevel provides 5 pre-built funnel templates that can be added to any account with one click. Each template includes pre-designed pages and pre-written copy that you customize for your business.

**How to Use:**
1. Click the template link
2. Click "Add funnel" in HighLevel
3. Customize to your business using the prompts below

> ⚠️ **Important:** Do NOT use the same words from the original sales copy/testimonials. Use the template structure and design, but personalize all copy for your offer.

---

## Template 1: Book a Call Funnel

### Purpose
Book appointments or calls with potential clients.

### Page Flow
```
Opt-in Page → Booking Page → Confirmation Page
```

### Page Details
| Page | Content |
|------|---------|
| **Opt-in** | Headline, subheadline, benefits, CTA button |
| **Booking** | Calendar widget, time slots, form fields |
| **Confirmation** | Appointment details, what to expect, next steps |

### Best For
- Service businesses
- Consultants
- Coaches
- Agencies booking discovery calls

## Template 2: ChatGPT Opt-In Funnel

### Purpose
Capture leads by offering a free AI-related resource (ChatGPT prompts, strategies, etc.).

### Page Flow
```
Opt-in Page → Delivery/Thank You Page
```

### Page Details
| Page | Content |
|------|---------|
| **Opt-in** | Headline about AI/ChatGPT, resource description, name + email form |
| **Delivery** | Download link, next steps, upsell opportunity |

### Best For
- Lead generation
- Building email list
- AI/tech audiences

### Original Headline Template
```
200+ FREE AI SEO Strategies & VIDEO COURSE -
Steal The EXACT SEO Strategies That 2X'd My Traffic in 11 Days!
```

## Template 3: Mastermind Funnel

### Purpose
High-ticket mastermind or premium program application.

### Page Flow
```
Application Page → Booking/Interview → Payment → Welcome
```

### Page Details
| Page | Content |
|------|---------|
| **Application** | Multi-step form, qualification questions, social proof |
| **Booking** | Calendar for interview call |
| **Payment** | Payment form for mastermind fee |
| **Welcome** | Onboarding info, next steps, community access |

### Best For
- High-ticket coaching
- Mastermind groups
- Premium communities
- VIP programs

---

## Template 4: Course Tripwire Funnel

### Purpose
Low-cost course or digital product with upsell to premium offer.

### Page Flow
```
Sales Page → Checkout → Upsell Page → Thank You
```

### Page Details
| Page | Content |
|------|---------|
| **Sales** | Course description, benefits, testimonials, pricing |
| **Checkout** | Order form, payment details |
| **Upsell** | Premium offer (one-time offer), bonus stack |
| **Thank You** | Access details, next steps |

### Best For
- Online course creators
- Digital product sellers
- Low-ticket to high-ticket funnels

---

## Template 5: Free Book Funnel

### Purpose

### Page Flow
```
Opt-in Page → Download/Thank You Page
```

### Page Details
| Page | Content |
|------|---------|

## Customization Prompts for All Templates

```

Steal The EXACT SEO Strategies That 2X'd My Traffic in 11 Days!

```

### Prompt 2: Alex Hormozi Style Headlines
