# Landing Pages & Websites — SOP

> Standard Operating Procedure for building landing pages, opt-in pages, and websites with AI.
> Source: Julian Goldie SEO — Skool Post #6
> Related: [[highlevel-funnels-reference]] | [[landing-pages-prompts]] | [[lead-connector-derivatives-comparison]]

---

## Overview

This SOP covers the complete process for building high-converting landing pages, opt-in funnels, and full websites using AI tools (primarily Claude) and Lead Connector derivative platforms (GoHighLevel, GoStackBase, BuildwithOS).

---

## Phase 1: Planning

### Step 1: Define Your Offer
Before building anything, clarify:
- What are you selling/promoting?
- Who is the target audience?
- What is the desired action? (book a call, opt in, buy, download)
- What's the value proposition?

### Step 2: Choose Your Platform

| Platform | Best For | Starting Price |
|----------|----------|---------------|
| **GoHighLevel** | Full marketing suite, agencies | $97/mo |
| **GoStackBase** | Digital products, client portals | $79/mo |
| **BuildwithOS** | AI-first, project management | $97/mo |

### Step 3: Choose Your Template
Select a proven template from your platform:
- **Book a Call** — for booking appointments
- **ChatGPT Opt-In** — for lead capture with AI resource
- **Mastermind** — for high-ticket applications
- **Course Tripwire** — for low-cost upsell funnels
- **Free Book** — for lead magnet delivery

---

## Phase 2: Build the Landing Page

### Step 4: Set Up the Page Structure
1. Log into your chosen platform
2. Create a new funnel/landing page from template
3. Set up the page flow:
   - **Opt-in pages:** Form → Thank You/Confirmation
   - **Sales pages:** Sales → Checkout → Upsell → Thank You
   - **Funnels:** Opt-in → Sales → Checkout → Delivery

### Step 5: Design with HighLevel (or Equivalent)
1. Use the platform's drag-and-drop builder
2. For GoHighLevel: Use the visual funnel builder
3. For GoStackBase: Use the landing page builder
4. For BuildwithOS: Use pre-built business kit templates

> **Pro Tip:** Don't automate the design setup with AI — use the platform's templates. AI is for content generation only.

### Step 6: Connect Your Domain
1. Set up root domain or subdomain
2. Configure DNS settings
3. Enable SSL (usually automatic)

---

## Phase 3: Generate Content with AI

### Step 7: Generate Headline & Subheadline
Use Claude (recommended) to generate compelling copy.

**Prompt 1: Rewrite for Your Offer**
```
Rewrite this headline and subheadline:
"200+ FREE AI SEO Strategies & VIDEO COURSE -
Steal The EXACT SEO Strategies That 2X'd My Traffic in 11 Days!"
But personalize this for my offer: [INSERT YOUR OFFER HERE]
```

**Prompt 2: Alex Hormozi Style Variations**
```
OK now give me 10 more, Alex Hormozi style
```

**Prompt 3: Rewrite Body Sections**
```
Now rewrite this section: [INSERT SECTION TEXT]
```

### Step 8: Generate Full Page Content
1. Start with headline + subheadline (Prompt 1)
2. If you don't like the first result, get 10 more variations (Prompt 2)
3. Rewrite each body section individually (Prompt 3)
4. Review and edit for brand voice

> **Best Practice:** Always use Claude for writing content. Don't rewrite all content at once — do it in parts (headline first, then each section).

---

## Phase 4: Set Up Automations

### Step 9: Configure Email Delivery
1. Set up your sending domain
2. Create the opt-in confirmation email
3. Create the resource delivery email (for lead magnets)
4. Set up the nurture sequence

### Step 10: Configure CRM & Pipelines
1. Create a pipeline for your offer
2. Set up automation triggers:
   - Form submission → Add to pipeline
   - Form submission → Send email
   - Purchase → Move to "Customer" stage
   - No action after 3 days → Send follow-up

### Step 11: Connect Email Marketing Platform
1. Connect your email service (Mailchimp, ActiveCampaign, etc.)
2. Set up automation workflows
3. Create email sequences for:
   - Welcome series
   - Abandoned cart
   - Post-purchase follow-up

---

## Phase 5: Build a Full Website (Auto Blogging Method)

### Step 12: Set Up Auto Blogging
1. Create a blog structure
2. Use AI to generate SEO-optimized blog posts
3. Set up publishing schedule
4. Configure on-page SEO (meta tags, headers, internal links)

### Step 13: Build Website Pages
1. Home page
2. About page
3. Services/Products page
4. Blog page
5. Contact page
6. Privacy policy + terms

### Step 14: Optimize for SEO
1. Keyword research for each page
2. On-page SEO optimization
3. Internal linking structure
4. Submit sitemap to Google

---

## Phase 6: Test & Launch

### Step 15: Test Everything
1. Test form submissions
2. Verify email delivery
3. Check all links
4. Test on mobile devices
5. Test checkout flow (if selling)

### Step 16: Launch & Monitor
1. Publish the page/funnel
2. Set up tracking (Google Analytics, platform analytics)
3. Monitor conversion rates
4. A/B test headlines, CTAs, and layouts

## Alternative Platforms to HighLevel

If you don't have HighLevel, use these alternatives:

| Platform | Pros | Cons |
|----------|------|------|
| **GoStackBase** | Cheaper ($79/mo), client portals | No AI agents, no phone system |
| **BuildwithOS** | AI agents, phone calls, PM built-in | Different paradigm (not traditional funnels) |
| **Lovable** | Easy setup, AI-powered | Less feature-rich |
| **Bolt DIY** | Powerful, flexible | Requires more technical skill |
| **Replit** | Full control, customizable | Requires coding knowledge |

---

## Key Principles

1. **Use templates** — Don't build from scratch
2. **Claude for content** — Best AI for copywriting
3. **Part by part** — Don't rewrite everything at once
4. **Test before launch** — Always verify forms, emails, links
5. **Don't copy competitors** — Use templates but personalize everything
6. **More context = Better results** — Give AI detailed information about your offer
