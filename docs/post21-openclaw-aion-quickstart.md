## The Three Tools Explained

### What is Hermes?

Hermes is an **AI agent** — a smart robot assistant that lives on your computer. You give it a task, it figures out how to complete it step by step. It can write code, manage files, run workflows, and do things that would take you hours.

The problem? It normally lives inside a **Terminal** — a black screen with blinking text. Ugly, confusing, and not built for managing multiple robots at once.

### What is OpenClaw?

OpenClaw is another AI agent — similar to Hermes but with its own special powers. Built on top of Claude (Anthropic's AI), designed for coding and automation tasks. It connects to **MCP tools** — plugin add-ons that give your AI robot extra abilities. Think of it like adding apps to your phone, but for your AI agent.

### What is Aion UI?

Aion UI is the **dashboard** that brings everything together. Free and open source. Instead of a scary terminal screen, you get a clean, beautiful chat interface. You can run Hermes, OpenClaw, Claude Code, Gemini CLI, and more — all in one place. Switch between them instantly. Schedule tasks. Control from your phone.

## Key Features

### Project Folders + Context Mode

When you open a folder inside Aion UI, your AI agent **only focuses on that project**. It reads every file, understands what you're building, and gives answers specific to YOUR work — not generic internet answers.

### Scheduled Tasks Across ALL Agents

Schedule tasks across EVERY agent you have running. Set the task name, description, which agent handles it, and when it runs. Your AI team works while you sleep.

### Skills Marketplace

Skills are **pre-built superpowers** you install into your agents. Need SEO? Coding? Content? Search the marketplace, install the skill, and your agent gets smarter instantly.

### Voice Input

Aion UI supports **speech-to-text** — just talk and it transcribes. Your AI agent hears your instruction and gets to work.

### Image Models

Add image generation models inside Aion UI. Configure in model settings, and your AI agents can generate visuals as part of their workflow.

## SOP — Setup Procedure

### Phase 1: Install Aion UI

1. Go to the [Aion UI GitHub page](https://github.com/iOfficeAI/AionUi)
2. Scroll down to **Latest Releases**
3. Download the correct version for your system (Mac, Windows, or Linux)
4. Open the installer file
5. Click **Open** when prompted
6. Aion UI launches — you'll see the main dashboard

✅ Installation complete.

### Phase 2: Connect Your Agents

7. Inside Aion UI, go to **Settings**
8. Navigate to the **Assistants or Agents** section
9. Add **Hermes** as your first agent (must be installed on your machine)
10. Add **OpenClaw** as your second agent
11. Add **Claude Code** if you have it installed
12. Verify each agent by sending a test message (e.g. "Are you working?")
13. Confirm you see a response from each agent

✅ Agents connected.

### Phase 3: Set Up a Project Folder

14. Click **New Chat** inside Aion UI
15. Select your agent (e.g. Hermes)
16. Switch from **Chat Mode** to **Folder Mode**
17. Navigate to the folder holding your project files
18. Select that folder as your context
19. Your agent now has full awareness of everything inside that folder

✅ Project context active.

### Phase 4: Install Skills

20. Go to the **Skills Marketplace** inside Aion UI
21. Search for skills relevant to your work (SEO, coding, content, etc.)
22. Click **Install** on the skills you want
23. Assign installed skills to the relevant agents

✅ Skills deployed.

### Phase 5: Schedule Your First Automated Task

24. Go to the **Scheduled Tasks** section
25. Click **Add New Task**
26. Enter a task name (e.g. "Daily SEO Report")
27. Write a clear task description and prompt
28. Select which agent handles this task
29. Set the time and frequency
30. Make sure **Keep Awake** is turned on
31. Save and confirm

✅ Automation live.

### Phase 6: Set Up Voice Input (Optional)

32. Go to **Settings → Capabilities → Speech to Text**
33. Select your voice input provider
34. Configure and save
35. Click the mic icon inside the chat to speak your instructions

✅ Voice mode activated.
