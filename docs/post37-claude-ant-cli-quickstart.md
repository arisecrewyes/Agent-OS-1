## What Is the Ant CLI?

The **ANT CLI** (Anthropic's agent CLI tool) lets you build **AI workers (agents)** that run on their own in the **cloud** — no babysitting required. Launched **April 8th, 2026**. Already used by companies like **Notion, Asana, and Rakuten**.

**Core idea:** Chatting with AI is like ordering one coffee at a time. The ANT CLI is like a pipe pumping coffee into a whole factory — your AI workers run independently, even when your laptop is off.

**You don't need to code.** You type plain commands, or ask Claude Code to run the ANT CLI for you in normal English.

---

## The Cloud Crew Stack (Framework)

A 4-layer framework for building your AI worker team:

| Layer | Name | What It Does |
|-------|------|-------------|
| 1 | 🧑‍💼 **The Worker** | One AI agent set up to do one job really well (emails, blog posts, etc.) |
| 2 | 🏠 **The Home** | Where the worker lives in the cloud so it can run on its own |
| 3 | 🧠 **The Memory** | Files and notes you give your worker so it knows your business (style, rules, past work) |
| 4 | 👁️ **The Watch** | How you check what your worker did, step by step, to make sure it did a good job |
