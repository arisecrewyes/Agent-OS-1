# 2026-06-25 — Skool Post #2: Infinite Context Engine

## Task
Install Julian Goldie's "Infinite Context Engine" (Skool Post #2) on the Hostinger VPS.
This is the OMI + Obsidian + Hermes Agent stack adapted for VPS deployment.

## What Was Built

### Docker Project: goldie-stack
**Location:** `/root/agentos/agent-projects/compose/goldie-stack/`

**Containers:**
1. **goldie-obsidian-vault** (port 8200) — Obsidian Vault web GUI + REST API
   - Custom FastAPI server with web-based Markdown editor
   - Vault stored in shared Docker volume `obsidian-vault-data`
   - Traefik route: `obsidian.srv1121935.hstgr.cloud`

2. **goldie-memory-engine** (port 8100) — OMI equivalent
   - LLM-powered memory extraction (using OpenRouter)
   - Auto-categorizes memories: goal, preference, project, task, person, habit, fact, quote
   - Saves to Obsidian vault via filesystem
   - Model: `openrouter/owl-alpha` (free)
   - Traefik route: `memory.srv1121935.hstgr.cloud`

3. **hermes-agent-7llb-hermes-agent-1** (ports 4860, 8642) — Hermes Agent
   - Updated with OpenRouter API key
   - Connected to Obsidian vault via volume mount
   - Dashboard on port 4860, API on port 8642
   - Traefik route: `hermes.srv1121935.hstgr.cloud`

4. **agentos-connector** (port 8888) — Agent OS Dashboard
   - Updated with vault volume mount (read-only)
   - Now reads from same Obsidian vault

### Key Files
