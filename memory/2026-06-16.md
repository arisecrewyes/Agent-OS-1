## Agent-to-Agent Communication System

### How It Works
- Each specialist agent (Content Creator, Memory & Brain, Skill Master) has a registry entry with keywords
- When a user sends a message to any agent, the chat API checks if the message matches any specialist's keywords
- If matched, a collaboration notice is injected into the system prompt
- The current agent can then acknowledge the specialist and delegate the task
- This mimics employees from different departments collaborating

### Specialist Agents
1. **Content Creator** (content-creator): 18 repos for video, media, AI gen, social scraping, sales
2. **Memory & Brain** (memory-brain): 11 repos for memory, knowledge extraction, vector search
3. **Skill Master** (skill-master): 11 repos for skills, optimization, conversion

### Built-in Agents (in sidebar)
- Agent Creator, OpenClaw, Hermes, Claude (original 4)
- Content Creator, Memory & Brain, Skill Master (new 3)
- All appear in the agents section with status badges
- Custom agents can still be added via "Add Agent" button
- Pin, reorder, and remove functionality preserved
