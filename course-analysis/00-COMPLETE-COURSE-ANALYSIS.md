### 5. Pre-Launch Testing Checklist (Complete QA SOP)

**Section 1: Setup Verification (Blockers 🔴)**
- API key validated & working
- Correct model (Claude Sonnet 4.5)
- Auto-recharge enabled
- System prompt loaded correctly
- Calendar integration connected

**Section 2: Channel Connectivity**
- SMS: send/receive, A2P active, STOP works
- Email: send/receive, not in spam
- FB/IG/WhatsApp: send/receive, under 30s

**Section 3: Conversation Testing (all 🔴 must pass)**
- Basic inquiry, Price question, Book appointment (full flow)
- Lead qualification → correct path, Disqualified lead → handled
- After hours message, Emergency/urgent request
- Multiple questions in one message, Typos understood
- Angry customer, Competitor mentioned
- "I want to speak to a human", Off-topic question

**Section 4: Quality Check**
- No hallucinations, Accurate info only, On-brand
- Grammar correct, Not too verbose, Maintains context

**Section 5: Automations & Follow-Ups**
- Pipeline stages, Follow-up timing, Appointment confirmations
- Escalation notifications, Lead routing

**Section 6: Go-Live Checklist**
- Test data removed, Client training done, Client approval received
- Admin access provided, Support docs delivered, Monitoring enabled

**Post-Launch Monitoring:**
- First 24h: Monitor every conversation, hourly response checks, verify bookings, review errors, track tokens
- First week: Daily performance review, client feedback, optimization opportunities
