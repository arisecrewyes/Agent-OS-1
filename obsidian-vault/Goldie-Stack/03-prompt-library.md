# Goldie Stack — AI Prompt Library

> Part of the [[00 - Index|Goldie Stack]] knowledge base.

## Knowledge Management

### Create a Knowledge Note
```
Create a note in my Obsidian vault about [topic].
Save it to [folder] with today's title.
Include: summary, key points, and action items.
```

### Research & Store
```
Research [topic] thoroughly and create a comprehensive
note in my Obsidian vault. Include:
1. Executive summary
2. Key findings (5-7 points)
3. Actionable insights
4. Related topics to explore
5. Sources and references
```

### Daily Memory Capture
```
Create today's daily note in my Obsidian vault with:
1. What I worked on today
2. Key decisions made
3. Ideas to explore
4. Tasks for tomorrow
```

## Daily Briefing (Jarvis)

### Generate Daily Briefing
```
Read my Obsidian vault and generate a daily briefing:
1. Open to-dos
2. What I worked on yesterday
3. Recent memory captures
4. Today's top 3 priorities
5. Upcoming events or deadlines
```

### Generate Weekly Briefing
```
Read my Obsidian vault and generate a weekly briefing:
1. Wins from this week
2. Day-by-day activity summary
3. Key lessons learned
4. Priorities for next week
5. Items that need follow-up
```

### Spoken Briefing
```
Generate my daily briefing and speak it in Jarvis voice.
Keep it concise — under 2 minutes.
Start with a greeting and today's date.
```

## Agent Integration

### Ask Agent to Reference Vault
```
Look in my Obsidian vault for information about [topic].
Then help me [task based on that information].
```

### Cross-Agent Knowledge Sharing
```
Save the results of our conversation about [topic]
to my Obsidian vault so other agents can access it.
```

## Vault Organization

### Create Folder Structure
```
Set up this folder structure in my Obsidian vault:
- Daily Notes
- Projects/[project name]
- Knowledge Base
- Meeting Notes
- Ideas
- Templates
- Archive
```

### Weekly Review
```
Review this week's daily notes in my Obsidian vault
and create a weekly summary with:
1. Accomplishments
2. Challenges
3. Key learnings
4. Next week's focus
```

## Tips

1. Be specific about vault paths
2. Use consistent naming (daily notes as `YYYY-MM-DD`)
3. Reference existing notes — build on what's there
4. Keep notes focused — one topic per note
5. Review regularly — have an agent organize monthly
