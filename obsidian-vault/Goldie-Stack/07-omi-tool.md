## What Is OMI?

OMI is a free app that runs in the background of your computer:
- **Listens** through your microphone
- **Watches** your screen
- **Takes notes** on everything you do, say, and work on
- **Organises** notes into clean, readable memories
- **Exports** memories to Obsidian

## Setup

1. Go to [omi.me](https://omi.me) and download
2. Create free account
3. Grant microphone + screen recording access
4. Let it run for at least one full working day
5. Connect to Obsidian via Memories → Export

## Key Features

- Background operation — silent, no interaction needed
- Audio + screen capture
- Automatic organisation into structured memories
- Obsidian export
- Optional: chat directly with OMI (paid, not required)

## In The Stack

OMI = Layer 1 (Capture) + Layer 2 (Organise) of the Infinite Context Engine™.

## For VPS Users

OMI is **not needed on the VPS**. The Obsidian vault is directly mounted as a Docker volume. Memory capture happens via manual note-taking + agent logs.
