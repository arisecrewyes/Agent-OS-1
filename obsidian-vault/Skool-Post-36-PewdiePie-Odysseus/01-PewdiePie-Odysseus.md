# 4th June: PewDiePie + Odysseus

## Summary

PewDiePie (24-year-old gamer with 110M+ subscribers) launched **Odysseus** on May 31st 2026 — a free, open-source AI workspace that runs on your own computer. It hit 30,000 GitHub stars in two days. This post covers what it is, a 4-layer framework for using it (The Odysseus Operator System), a 30-day implementation plan, and 100+ copy-paste prompts.

## Key Takeaways

- **Own your AI** — stop renting from big companies; Odysseus lives on your machine
- **4 layers:** Brain → Chatter → Operator → Memory
- **You're early** — the tool is days old; nobody has years of experience
- **Safety first** — check what the Operator can touch before letting it work
- **30-day plan** — every day has a free tool, a link, and an exact prompt
- **Community accountability** — building alongside others prevents quitting

## The Odysseus Operator System (4 Layers)

### Layer 1 — The Brain 🧠
The AI model that does the thinking. Choose local or cloud (via OpenRouter free key). Beginners should use a cloud brain to keep their laptop cool.

### Layer 2 — The Chatter 💬
Everyday helper for questions and writing. Like ChatGPT but private.

### Layer 3 — The Operator 🤖
Autonomous agent that plans steps, executes them, and returns results. This is where the real leverage is.

### Layer 4 — The Memory 📚
Remembers your notes, documents, and past chats. No need to explain yourself repeatedly.

## 30-Day Odysseus Operator Plan

### Week 1 — Foundation 🏗️
- **Day 1:** Visit [Odysseus page](https://pewdiepie-archdaemon.github.io/odysseus/) — just read, don't install
- **Day 2:** Install via Claude Code/Hermes — paste [GitHub link](https://github.com/pewdiepie-archdaemon/odysseus) and ask for walkthrough
- **Day 3:** Open `http://localhost:7000`, create admin account
- **Day 4:** Add free [OpenRouter](https://openrouter.ai) key as cloud brain
- **Day 5:** **Safety check** — review and toggle off unfamiliar tools
- **Day 6:** Chat about a trending Google Trends topic
- **Day 7:** Rest & review

### Week 2 — Automate 🤖
- **Day 8:** First Operator task — research 3 newest free AI tools
- **Day 9:** Schedule a daily research task
- **Day 10:** Build a neon dashboard in [CodePen](https://codepen.io)
- **Day 11:** Drop documents into Library, ask for themes
- **Day 12:** Deep research on beginner frustrations
- **Day 13:** Compare two AI brains side by side
- **Day 14:** Review

### Week 3 — Scale 📈
- **Day 15:** Turn frustration list into 5 social posts
- **Day 16:** Create talking-avatar video via [HeyGen](https://www.heygen.com)
- **Day 17:** Upload doc to [NotebookLM](https://notebooklm.google.com) for summary
- **Day 18:** Save best prompts as reusable skills
- **Day 19:** Build animated bubble page in CodePen
- **Day 20:** Connect email helper, summarize unread
- **Day 21:** Review

### Week 4 — Systematise ⚙️
- **Day 22:** Write your own playbook in [Google Docs](https://docs.google.com)
- **Day 23:** Combine scheduled tasks into morning briefing
- **Day 24:** "Explain my setup in 5 sentences"
- **Day 25:** Back up important files
- **Day 26:** Fun CodePen build — retro arcade score screen
- **Day 27:** Teach one thing in the community
- **Day 28:** Plan next 3 automation tasks
- **Day 29:** Reflect on progress
- **Day 30:** Show off — you own a full AI workspace

## The 7 Belief Shifts

| Old Belief | New Belief |
|-----------|-----------|
| I'm not technical enough | I just need to find the next stone |
| I need an expensive computer | I can run it on the computer I already own |
| Open-source means dodgy | Open-source = transparent, and I'm in control |
| Big companies will always be better | Owning my setup matters more than renting the fanciest one |
| I've missed the boat | I'm early — showing up daily is how I win |
| I won't know what to do | I just need today's one small step |
| I'll quit in a week | I need people building alongside me |
