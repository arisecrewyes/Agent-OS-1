# OpenClaw — AI Prompt Library

> Part of the [[00 - Index|OpenClaw 6 Hour Course]] knowledge base.

## Content Creation

### SEO Blog Post
```
Create an SEO-optimized blog post about [topic],
include keyword [keyword] 5-7 times naturally,
add internal links to [your site],
and publish to WordPress
```

### YouTube Thumbnail
```
Create a YouTube thumbnail about [topic]
```

### AI Avatar Video
```
Create an AI avatar video about [topic] using HeyGen
```

### Voice Note
```
Create a voice note in my voice about [topic]
```

### Landing Page
```
Create a landing page for [business],
make it conversion-optimized,
deploy to Netlify at [domain]
```

## Research & Monitoring

### YouTube Competitor Monitoring
```
Monitor these competitor YouTube channels: [URLs].
Alert me when they publish a video that performs 2x better
than their average. Check every 60 minutes.
```

### Web Search
```
Using the Chrome extension, go to Google and search for [query].
Give me the top 3 results with summaries.
```

## Social Media

### X/Twitter Post
```
Create a post about [topic] and post to X.
Include 3 relevant hashtags and an image.
```

## Productivity

### Weekly YouTube Analysis
```
Every Monday at 8am, analyze my YouTube performance
from last week and send me a report
```

### Daily Briefing
```
Every morning at 9am, send me a briefing with:
1. Today's calendar events
2. Urgent emails
3. Weather for my location
4. Top 3 priorities for today
```

## Workflows

### Complete SEO Workflow
```
Research keyword → Generate outline → Write 1500-word post →
Optimize for SEO → Add internal links → Publish to WordPress →
Create social posts → Share on Twitter/X
```

### Content Pipeline
```
Watch folder: /content/drop
When new files appear, scan them and group by topic.
Each morning, send me the top 3 content ideas.
```

### Lead Management
```
Scan inbox every 30 min for new leads.
Extract: name, company, need, timeline, budget.
Add to Google Sheet. Draft response.
Send me Telegram with lead details.
I approve, it sends.
```
