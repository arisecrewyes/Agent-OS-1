## 📋 Complete Setup SOP

### Phase 1 — Set Up OpenRouter (10 minutes)
1. Go to openrouter.ai
2. Click "Sign Up" (Google or email)
3. Profile icon → "API Keys" → "Create Key"
4. Name it "Hermes Agent Key" → Copy and save
5. Search "Owl Alpha" → Click on `openrouter/owl-alpha`
6. Confirm $0/M input and output tokens
7. Note model string: `openrouter/owl-alpha`

### Phase 2 — Install Hermes Agent (15 minutes)
1. Go to Hermes Agent website (Nous Research)
2. Follow installation for your OS
3. Open Hermes Agent → Settings/Model Configuration
4. Select OpenRouter as provider
5. Paste API key
6. Model field: `openrouter/owl-alpha`
7. Save and test connection

### Phase 3 — Run First Agent Task (10 minutes)
1. Think of one weekly task (30+ min)
2. Open Hermes Agent
3. Type task in plain English
4. Watch what it does
5. Review output, refine, save as template

### Phase 4 — Build First Workflow (30 minutes)
1. Pick a multi-step process (research → write → publish)
2. Break into steps
3. Create prompts for each step
4. Run in sequence, feeding outputs
5. Save as workflow
6. Schedule automation

### Phase 5 — Scale (Ongoing)
1. Add one new workflow per week
2. Document what works
3. Share wins for feedback
4. Stack workflows
5. Audit every 30 days
