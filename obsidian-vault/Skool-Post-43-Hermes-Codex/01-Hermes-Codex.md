## Overview

The Goldie Stack Framework™ — 4-layer system: Hermes Agent (Brain) + MCP (Hands) + Codex (Builder) + Business Automation (Output).

## 4 Layers

1. **Brain** — Hermes Agent: talks to tools, reads files, writes code, sends messages
2. **Hands** — MCP: bridge to GitHub, Stripe, databases, hundreds of tools
3. **Builder** — Codex: OpenAI coding agent, reads/writes/fixes code automatically
4. **Output** — Business Automation: SEO, content, outreach, code all automated

## 5 Principles

1. Connect Everything
2. Automate Boring Stuff First
3. Agents Compound
4. Build Once, Profit Forever
5. Stay In Loop, Get Out Of The Way

## Setup

1. Install Hermes Agent
2. Install Codex CLI: `npm install -g @openai/codex`
3. Connect via MCP config (one config file, 30 min setup)
4. Verify connection
5. Automate
