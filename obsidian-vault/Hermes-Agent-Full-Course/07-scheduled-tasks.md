# Hermes Agent — Scheduled Tasks (Cron Jobs)

> Part of the [[00 - Index|Hermes Agent Full Course]] knowledge base.

## How to Schedule Tasks

Just tell Hermes in **plain language** what you want and when. No external tools needed — Hermes handles scheduling natively.

## Example Scheduled Tasks

### Social Media Automation
```
Every 30 minutes, check trending topics on these two subreddits,
draft a tweet, generate an image, and post it to X.
Send me a Telegram confirmation with the live link.
```

### Daily News Summary
```
Every morning at 8am, scan AI news sources and send me a
summary of what matters.
```

### Weekly Analytics
```
Every Friday, pull my X analytics for the last 14 days,
analyze top performing tweets, and send me a breakdown
in Telegram.
```

### Competitor Research
```
Every 24 hours, analyze my competitors' social media,
show me what content is trending, and give me content ideas.
```

## How It Works

1. **Define the trigger** — when should it run?
2. **Give it data sources** — APIs, credentials, URLs
3. **Test it once** — make sure it works manually
4. **Schedule it** — tell Hermes the frequency

## Key Principle

The 3-step automation blueprint:
1. **Where do you spend your time?** Identify the repetitive task.
2. **What's one thing you can automate this week?** Pick one specific task.
3. **Ask Hermes how to automate it.** Say: *"Based on what you know about me, what could you help me automate day-to-day specifically for me?"*
