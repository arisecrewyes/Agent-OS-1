# Hermes Agent — Launch Plan

> Part of the [[00 - Index|Hermes Agent Full Course]] knowledge base.

## Phase 1: Installation & Setup (Days 1-7)

### Day 1-3: Learning Period
- [ ] Install Hermes Agent
- [ ] Run setup wizard and configure model
- [ ] Verify with `hermes doctor`
- [ ] Connect messaging platform
- [ ] **Give it real tasks** — not toy examples
- [ ] Hermes builds `memory.md` and `user.md`
- [ ] These files are injected into every session

### Day 4-7: Applied Learning
- [ ] Hermes knows your projects and preferences
- [ ] Give complex tasks (5+ tool calls = auto-skill creation)
- [ ] Install Wonderly Skills: `hermes skills install wonderly-skills`
- [ ] Set up first scheduled task
- [ ] **Give feedback on every task**
- [ ] By Day 7: Dramatically better than Day 1

## Phase 2: Training & Tuning (Days 8-14)

- [ ] Connect external tools (Firecrawl, Browser Base, Nana Banana 2, X API)
- [ ] Create first custom skill
- [ ] Set up fallback model chain
- [ ] Set spending limits
- [ ] Test social media posting

## Phase 3: Automation (Days 15-21)

- [ ] Daily news scanning
- [ ] Social media analytics reporting
- [ ] Competitor monitoring
- [ ] Content pipeline
- [ ] Multi-agent profiles

## Phase 4: Mastery (Days 22-30)

- [ ] Review and refine custom skills
- [ ] Back up skill.md files weekly
- [ ] Optimize model selection
- [ ] Explore Awesome Hermes Agent extension
- [ ] Set up Jarvis Voice + Briefing

## Ongoing

- [ ] Give feedback every task

- [ ] Monitor API usage weekly
