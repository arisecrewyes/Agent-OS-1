# Hermes — One-Click Setups

> Two easy one-click methods to install and run Hermes (Ollama + MaxHermes).
> Source: Julian Goldie SEO — Skool Post #5
> Related: [[14 - QuickStart Guide]] | [[00 - Index]]

---

## Option 1: Ollama (Recommended)

### Install
```bash
curl -fsSL https://ollama.com/install.sh | sh
```

### Launch Hermes
```bash
ollama launch hermes
```

### Benefits
- ✅ Free models locally
- ✅ Cloud models
- ✅ Setup Hermes in 1 click
- ✅ Easily switch between models

---

## Option 2: MaxHermes

### Benefits
- ✅ Easy to set up
- ✅ Non-technical friendly
- ✅ Scheduled tasks work
- ✅ All tools work
- ✅ Creates images/videos out of the box

### Drawbacks
- ⚠️ Not as good as local
- ⚠️ Can be buggy
- ⚠️ Less control
