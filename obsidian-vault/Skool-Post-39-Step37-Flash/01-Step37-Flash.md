## Overview

Step 3.7 Flash (by StepFun) — free for 30 days inside Hermes Agent. 400 words/second, vision, web search, coding, automation.

## The LIFT Method™

- **Learn** — Spend a few minutes understanding the tool
- **Install** — Get it running
- **Find** — Pick ONE boring task for AI to take over
- **Teach** — Once it works once, teach it to repeat

## Setup

1. Install Hermes Agent from GitHub
2. Get OpenRouter API key (free)
3. Connect Step 3.7 Flash in Settings → Model
4. Optionally link Telegram/Discord/WhatsApp
5. Test with a simple prompt

## Capabilities

- Vision (screenshots → websites)
- Web search (real answers)
- Browsing & scraping
- Code writing & execution
- Long-form writing
- Analysis of large reports
- Automation while you sleep

## Limiting Beliefs Busted
