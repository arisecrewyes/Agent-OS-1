# 2nd June: Claude Ant CLI

> Skool Post #37 — AI Profit Boardroom · New Daily Updates + Advanced Tutorials · 7:49

## Summary

The **ANT CLI** (Anthropic's agent CLI tool) lets you build **AI workers (agents)** that run on their own in the **cloud** — launched **April 8th, 2026**. Already used by **Notion, Asana, and Rakuten**. You don't need to code — just type plain commands or ask Claude Code to do it in English.

**Core shift:** Chatting with AI = ordering one coffee at a time. ANT CLI = a pipe pumping coffee into a whole factory.

---

## The Cloud Crew Stack (Framework)

| Layer | Name | What It Does |
|-------|------|-------------|
| 1 | 🧑‍💼 **The Worker** | One AI agent set up to do one job really well |
| 2 | 🏠 **The Home** | Where the worker lives in the cloud (runs independently) |
| 3 | 🧠 **The Memory** | Files/notes giving the worker your business context |
| 4 | 👁️ **The Watch** | Tracing — check every step your worker took |

## Empowering Beliefs Shift

| ❌ Old Belief | ✅ New Belief |
|--------------|--------------|
| I need to be a coder | I build with words, not code |
| This is too complicated | I can do what big companies do |
| Only for big tech | Being early is my advantage |
| No time to babysit AI | Build once, works forever |
| Tried AI before, didn't help | I'm hiring AI, not chatting |
| Might break something | I can see every step |
| Everyone's already doing it | I'm at the front of the line |

## Old Way vs New Way

| ❌ Old Way | ✅ New Way |
|-----------|-----------|
| One task at a time in chat | Cloud workers run independently |

| Hire engineers for servers | First worker in an afternoon |
