# OpenClaw + Aion UI (Post #21)

> Converted from Julian Goldie Skool Post #21 — 2nd May
> Original: https://www.skool.com/ai-profit-lab-7462/classroom/9daf24e1?md=5ee0ee76063a4b2fa897d5d39a2bb7d3

## The Core Idea

> "Most people are using AI agents the hard way. There's a better way. And it's completely free."

OpenClaw + Hermes + Aion UI gives you a **free, unified dashboard** to run multiple AI agents simultaneously. No terminal. No subscriptions. Just a clean chat interface where you control Hermes, OpenClaw, Claude Code, Gemini CLI, and more — all in one place.

## The Three Tools

| Tool | What It Is | Role |
|------|-----------|------|
| **Hermes** | AI agent | Head chef — does complex tasks autonomously |
| **OpenClaw** | AI agent (Claude-based) | Sous chef — coding and automation specialist |
| **Aion UI** | Free open-source dashboard | Kitchen manager — coordinates all agents |

### Hermes

An AI agent that lives on your computer. Give it a task in plain English, it figures out how to complete it. Can write code, manage files, run workflows. Normally trapped in Terminal (black screen, blinking text).

### OpenClaw

AI agent built on Claude (Anthropic). Designed for coding and automation. Connects to **MCP tools** — plugin add-ons that give extra abilities. Like adding apps to your phone, but for your AI agent.

### Aion UI

The dashboard that brings it together. Free, open source. Clean chat interface. Run Hermes, OpenClaw, Claude Code, Gemini CLI — all in one place. Switch instantly. Schedule tasks. Control from phone.

## Why It Matters

> "The difference between Terminal and Aion UI is like the difference between driving with no dashboard vs. flying a plane with a full cockpit."

**What you unlock:**

- 🤝 Multiple AI agents working simultaneously
- 📅 Scheduled tasks running 24/7
- 📁 Folder-based context (AI knows your project)
- 🎙️ Voice input (speech-to-text)
- 🧩 Skills Marketplace (install new abilities)
- 📱 Phone control (manage from anywhere)
- 🔀 Switch between agents in seconds

## Key Features Deep Dive

### Project Folders + Context Mode

Open a folder → AI focuses only on that project. Reads every file. Understands what you're building. Gives specific answers, not generic ones.

### Scheduled Tasks Across ALL Agents

Set task name, description, agent assignment, and schedule. Works across every agent you run. Your AI team works while you sleep.

### Skills Marketplace

Pre-built superpowers. Need SEO? Coding? Content? Search, install, done. Agent gets smarter instantly.

### Voice Input

Speech-to-text built in. Talk to your AI. Game changer for busy people.

### Image Models

Add image generation models in settings. Agents can generate visuals as part of workflows.

## SOP — 6-Phase Setup

### Phase 1: Install Aion UI

1. Go to [Aion UI GitHub](https://github.com/iOfficeAI/AionUi)
2. Scroll to **Latest Releases**
3. Download for your OS (Mac/Windows/Linux)
4. Open installer, click **Open**
5. Dashboard launches

### Phase 2: Connect Agents

6. Settings → Assistants/Agents
7. Add Hermes (must be installed)
8. Add OpenClaw
9. Add Claude Code (if available)
10. Test each with "Are you working?"

### Phase 3: Project Folder

11. New Chat → Select agent
12. Switch to **Folder Mode**
13. Navigate to project folder
14. Select as context

### Phase 4: Install Skills

15. Skills Marketplace → Search
16. Install relevant skills
17. Assign to agents

### Phase 5: Schedule Tasks

18. Scheduled Tasks → Add New
19. Name, description, agent, time
20. Enable **Keep Awake**
21. Save

### Phase 6: Voice (Optional)

22. Settings → Capabilities → Speech to Text
23. Select provider, configure
24. Use mic icon in chat

## Known Issues

- OpenClaw can be buggy — refresh/restart if errors occur
- Hermes and Claude Code are more stable inside Aion UI
- This is normal, not an Aion UI problem
